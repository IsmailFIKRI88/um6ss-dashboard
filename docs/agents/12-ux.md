# Agent : UX Designer — Visualisation de l'incertitude

> Session 2 · Mars 2026 · Mission : présenter des estimations sans qu'elles soient prises pour des faits

## Principe : deux registres visuels

| Élément | Fait mesuré | Estimation Fermi |
|---|---|---|
| Bordure KPICard | `3px solid ${color}` | `3px dashed #6B85A8` |
| Background | cardStyle normal | `#F0F4FA` (clair) / `rgba(65,140,203,0.06)` (sombre) |
| Couleur texte | colors.dark | `#6B85A8` (bleu-ardoise désaturé) |
| Valeur | `3 421` (exact) | `~23K` (arrondi + tilde) |
| Sous-titre | — | `18K — 28K` (fourchette, monospace) |
| Badge | — | `~ MODÈLE` (pill #E8EFF7, border #C5D5E8) |

## Représentation de l'incertitude

**Fourchette en deux lignes, pas ±** :
```
Ligne 1 (grande) : ~23 000
Ligne 2 (petite) : 18 000 — 28 000
```
Convention des instituts nationaux de statistiques. Lue naturellement sans formation.

**Sur les graphiques** : zone hachurée `repeating-linear-gradient(45deg, ...)` pour la fourchette, barre pleine pour la valeur centrale.

## Le funnel TAM → SAM → SOM → Inscrits

Barres horizontales en cascade. Les 3 lignes Fermi en style estimation (hachures). La dernière ligne "Inscrits réels" en style fait mesuré (barre pleine verte). L'asymétrie visuelle EST le message.

## Taux de pénétration

Barre horizontale (pas jauge circulaire — biais de perception angulaire). Zone d'incertitude SOM en fond hachuré. Trait vertical pour la valeur réelle. Couleurs : rouge < 50%, orange 50-79%, vert ≥ 80%.

## Disclaimers

Bandeau contextuel ancré en haut de la section Fermi, permanent mais discret :
- Bordure gauche 3px solid accent
- Background rgba accent 8%
- Bouton "Voir les hypothèses" (seul élément interactif)
- Ne dit jamais "données fausses" — dit "modèle reposant sur des hypothèses"
- Dismissable via sessionStorage (pas localStorage)

## Anti-patterns

1. **Fausse précision** : `~23K` pas `23 412` → utiliser `fmt.fermi()`
2. **Incertitude cachée** : fourchette obligatoire sur chaque KPI Fermi
3. **Surcharge cognitive** : révélation progressive (valeur → fourchette → hypothèses)
4. **Paralysie décisionnelle** : chaque section se termine par une conclusion opérationnelle
